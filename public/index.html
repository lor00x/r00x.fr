Proxy LDAP en Golang

Bonjour ! Comme promis dans mon article précédent je vais expliquer aujourd’hui la mise en application du langage Golang pour implementer le protocole LDAP. LDAP LDAP est un protocole permettant d’interagir avec un annuaire (LDAP = Lightweight Directory Access Protocol). Il permet à un client d’effectuer toutes sorte d’opérations standards auprès d’un serveur d’annuaire: authentification, interrogation, mise à jour et suppression d’entrées… De protocole, LDAP a ensuite évolué pour représenter une norme pour les systèmes d’annuaires, incluant un modèle de données, un modèle de nommage, un modèle fonctionnel basé sur le protocole LDAP, un modèle de sécurité et un modèle de réplication (source Wikipedia). »
