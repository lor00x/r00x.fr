Langage machine pour tous ! La programmation haut niveau (JAVA, C#, PHP) donne souvent une impression de facilité par rapport à des langages plus proches du système (assembleur, C, C++). C’est oublier que, même à haut niveau, votre code finira transformé en langage machine avant d’être exécuté. La structure de votre code va influencer le compilateur et le langage machine qu’il produit.
